# 固定遅延+差分補正MVDR 基本方式設計書

## 1. 目的

本方式は、固定整相処理を主経路として維持しつつ、MVDR によって得られる最適重みとの差分だけを補正 FIR として実装する方式である。

方式名を以下とする。

```text
固定遅延+差分補正MVDR
```

狙いは以下である。

- 整数遅延および小数遅延による固定整相をリアルタイム主経路に置く。
- MVDR 重み全体を時間領域 FIR 化するのではなく、固定整相重みとの差分だけを FIR 化する。
- 差分補正 FIR を 128 tap 程度に抑える。
- 統計推定は短い FFT によって逐次更新し、リアルタイム出力は時間領域 FIR で処理する。
- overlap-save は係数設計には使わず、リアルタイム補正 FIR は通常の時間領域畳み込みで実装する。

本方式は、数学的には GSC 形式の MVDR と等価な構造として解釈できる。ただし、blocking matrix を明示せず、固定整相重みと MVDR 重みの差分を直接補正枝として実装する点が特徴である。

---

## 2. 全体構成

### 2.1 リアルタイムルート

```text
信号入力 x[n]
  ├─ 固定遅延整相枝
  │    ├─ 整数遅延
  │    ├─ 小数遅延 FIR
  │    └─ 固定整相出力 y0[n]
  │
  └─ 差分補正枝
       ├─ 128 tap 補正 FIR
       └─ 補正出力 z[n]

最終出力：
y[n] = y0[n] - z[n]
```

### 2.2 統計ルート

```text
信号入力 x[n]
  → 128 sample block 化
  → 128点 FFT
  → 周波数ビンごとの共分散行列 R[k] を指数平均更新
  → 1秒ごとに MVDR 重みを計算
  → 固定整相重みとの差分 q[k] を計算
  → IFFT
  → 128 tap 差分補正 FIR 係数へ変換
  → リアルタイムルートの補正 FIR に反映
```

統計ルートは係数設計用であり、リアルタイム信号の周波数領域畳み込みを目的としない。

---

## 3. 記号定義

| 記号 | 意味 |
|---|---|
| \(M\) | チャネル数 |
| \(f_s\) | サンプリング周波数 |
| \(N_{\mathrm{fft}}\) | 統計推定用 FFT 点数 |
| \(L\) | 統計推定用ブロック長 |
| \(K\) | 周波数ビン数 |
| \(x[n]\in\mathbb{C}^{M}\) | 時刻 \(n\) の多チャネル入力 |
| \(X_b[k]\in\mathbb{C}^{M}\) | block \(b\)、周波数 bin \(k\) の多チャネル FFT ベクトル |
| \(a[k]\in\mathbb{C}^{M}\) | 目標方向のステアリングベクトル |
| \(R[k]\in\mathbb{C}^{M\times M}\) | 周波数 bin \(k\) の共分散行列 |
| \(w_0[k]\in\mathbb{C}^{M}\) | 固定整相重み |
| \(w_{\mathrm{MVDR}}[k]\in\mathbb{C}^{M}\) | MVDR 重み |
| \(q[k]\in\mathbb{C}^{M}\) | 差分補正重み |
| \(h_q[m,l]\) | チャネル \(m\)、tap \(l\) の差分補正 FIR 係数 |
| \(y_0[n]\) | 固定整相出力 |
| \(z[n]\) | 差分補正出力 |
| \(y[n]\) | 最終出力 |

本設計では周波数領域出力を以下で統一する。

\[
y[k] = w[k]^H X[k]
\]

ここで \((\cdot)^H\) は複素共役転置を表す。

---

## 4. 固定整相重み

目標方向のステアリングベクトルを \(a[k]\) とする。目標信号のみが存在する場合、観測ベクトルは

\[
X[k] = a[k]s[k]
\]

と表せる。

固定整相重み \(w_0[k]\) は、目標方向に対して歪みなし条件を満たすように設計する。

\[
w_0[k]^H a[k] = 1
\]

典型的には、ステアリングベクトルのノルムで正規化し、

\[
w_0[k] = \frac{a[k]}{a[k]^H a[k]}
\]

とする。

この固定整相重みは、時間領域実装では以下と等価に扱う。

```text
整数遅延フィルタ
  * 小数遅延フィルタ
  * チャネル合成
```

すなわち、周波数領域で見れば、固定整相枝は \(w_0[k]^H X[k]\) を計算しているものとみなす。

---

## 5. MVDR 重み

周波数 bin \(k\) における MVDR は、出力電力を最小化しつつ、目標方向のゲインを 1 に保つ重みである。

最適化問題は以下である。

\[
\min_{w[k]} w[k]^H R[k] w[k]
\]

subject to

\[
w[k]^H a[k] = 1
\]

解は、

\[
w_{\mathrm{MVDR}}[k]
=
\frac{R[k]^{-1}a[k]}{a[k]^H R[k]^{-1}a[k]}
\]

である。

実装では特異行列および条件数悪化を避けるため、対角ローディングを入れる。

\[
R_{\mathrm{load}}[k]
= R[k] + \epsilon[k] I
\]

したがって実際に用いる重みは、

\[
w_{\mathrm{MVDR}}[k]
=
\frac{R_{\mathrm{load}}[k]^{-1}a[k]}{a[k]^H R_{\mathrm{load}}[k]^{-1}a[k]}
\]

である。

対角ローディング量は、例えば以下で与える。

\[
\epsilon[k]
= \gamma \frac{\mathrm{tr}(R[k])}{M}
\]

ここで \(\gamma\) は対角ローディング係数である。

---

## 6. 差分補正重み

本方式では、MVDR 重みそのものを補正枝に使わない。

固定整相重みと MVDR 重みの差分を、差分補正重みとして定義する。

\[
q[k] = w_0[k] - w_{\mathrm{MVDR}}[k]
\]

リアルタイム処理では、固定整相出力から差分補正出力を引く。

\[
y[k]
= w_0[k]^H X[k] - q[k]^H X[k]
\]

ここに \(q[k]=w_0[k]-w_{\mathrm{MVDR}}[k]\) を代入すると、

\[
y[k]
= w_0[k]^H X[k]
- \left(w_0[k]-w_{\mathrm{MVDR}}[k]\right)^H X[k]
\]

\[
y[k]
= w_{\mathrm{MVDR}}[k]^H X[k]
\]

となる。

したがって、差分補正重みを完全に再現できる限り、本方式の最終出力は MVDR 出力と数学的に一致する。

---

## 7. GSC 形式との関係

通常の GSC は以下の形で表される。

\[
y[k] = w_0[k]^H X[k] - g[k]^H B[k]^H X[k]
\]

ここで \(B[k]\) は blocking matrix であり、目標方向成分を消す条件として

\[
B[k]^H a[k] = 0
\]

を満たす。

GSC 全体の重みは、

\[
w[k] = w_0[k] - B[k]g[k]
\]

である。

本方式では、

\[
q[k] = w_0[k] - w_{\mathrm{MVDR}}[k]
\]

を直接補正枝として使うため、明示的な \(B[k]\) と \(g[k]\) を持たない。

しかし、固定整相重みと MVDR 重みがともに歪みなし条件を満たすなら、

\[
w_0[k]^H a[k] = 1
\]

\[
w_{\mathrm{MVDR}}[k]^H a[k] = 1
\]

であるため、

\[
q[k]^H a[k]
= \left(w_0[k]-w_{\mathrm{MVDR}}[k]\right)^H a[k]
= 0
\]

となる。

すなわち、差分補正重み \(q[k]\) は目標方向を通さない補正枝として振る舞う。これは GSC の blocking branch と同等の意味を持つ。

したがって本方式は、

```text
blocking matrix を明示しない GSC 形式の MVDR
```

と解釈できる。

---

## 8. 共分散行列の推定

本方式では、統計ルートで短い FFT を連続実行し、周波数 bin ごとに共分散行列を指数平均する。

想定条件の例を以下に示す。

\[
f_s = 32768\ \mathrm{Hz}
\]

\[
N_{\mathrm{fft}} = L = 128
\]

1 block の時間長は、

\[
T_{\mathrm{block}}
= \frac{128}{32768}
= 3.90625\ \mathrm{ms}
\]

1 秒あたりの block 数は、

\[
\frac{32768}{128} = 256
\]

である。

各 block \(b\) で得られる FFT ベクトルを \(X_b[k]\) とすると、瞬時共分散は、

\[
\hat{R}_b[k]
= X_b[k]X_b[k]^H
\]

である。

指数平均による共分散更新は、

\[
R_b[k]
= \alpha R_{b-1}[k]
+ (1-\alpha)\hat{R}_b[k]
\]

とする。

忘却係数 \(\alpha\) は block 単位で設定する必要がある。有効時定数を \(T_R\) 秒とすると、

\[
\alpha
= \exp\left(-\frac{T_{\mathrm{block}}}{T_R}\right)
= \exp\left(-\frac{L}{f_s T_R}\right)
\]

である。

例として \(T_R=10\) 秒の場合、

\[
\alpha
= \exp\left(-\frac{128}{32768\cdot 10}\right)
\approx 0.99961
\]

である。

---

## 9. 128 tap FIR 化

差分補正重み \(q[k]\) は周波数領域の重みである。これを時間領域 FIR として実装するため、周波数 bin 方向に IFFT する。

チャネル \(m\) の差分補正周波数応答を \(q_m[k]\) とすると、時間領域係数は、

\[
h_q[m,l]
= \mathrm{IFFT}_k\{q_m[k]\}
\]

で得る。

リアルタイム補正枝は、

\[
z[n]
= \sum_{m=0}^{M-1}\sum_{l=0}^{L_q-1} h_q[m,l]^* x_m[n-l]
\]

または、実装上の共役規約に合わせて同等の形で計算する。

最終出力は、

\[
y[n] = y_0[n] - z[n]
\]

である。

ここで \(L_q=128\) とする。

注意すべき点は、IFFT によって得た FIR 係数が有限長 128 tap に完全に収まらない場合、本方式は完全な MVDR ではなく近似 MVDR になることである。したがって、128 tap 化後には再 FFT して、元の \(q[k]\) との差を確認する必要がある。

---

## 10. overlap-save を使わない理由

本方式では、統計ルートの FFT は共分散推定と係数設計のために使う。

リアルタイム出力は、

```text
固定整相枝：時間領域の整数遅延 + 小数遅延
差分補正枝：時間領域の 128 tap FIR
```

で処理する。

したがって、リアルタイム信号処理において overlap-save は必須ではない。

ただし、演算量が問題になる場合には、差分補正 FIR 部分だけを overlap-save 化する余地はある。本設計の初期実装では、方式確認を優先し、通常の時間領域 FIR とする。

## 11. 安定化設計

### 11.1 共分散の指数平均

共分散行列を時間方向に指数平均することで、以下の効果が得られる。

- スナップショット不足による不安定性を軽減する。
- 重み更新の時間変動を滑らかにする。
- 係数更新時の段差を抑制する。

ただし、指数平均だけでは特異行列を完全には防げない。

### 11.2 対角ローディング

MVDR 重み計算では、必ず対角ローディングを行う。

\[
R_{\mathrm{load}}[k]
= R[k] + \epsilon[k] I
\]

これにより、以下を防ぐ。

- 特異行列
- 条件数悪化
- 過大な null 形成
- 重みの過剰変動

### 11.3 係数更新

初期方式では、FIR 係数は 1 秒ごとに即時更新する。

係数段差は、時間方向に方位が変化する信号を入力し、BTR 上の縦縞または瞬間的なレベルジャンプとして確認する。

問題が出た場合は、以下の順で対策する。

1. 共分散時定数を長くする。
2. 周波数方向に MVDR 重みを平滑化する。
3. FIR 係数を時間方向に平滑化する。
4. 旧 FIR 出力と新 FIR 出力をクロスフェードする。

## 12. 評価指標

本方式の評価では、少なくとも以下を確認する。

### 12.1 数学的等価性

周波数領域で、

\[
w_0[k] - q[k] \approx w_{\mathrm{MVDR}}[k]
\]

が成り立つこと。

### 12.2 目標方向の歪みなし条件

固定整相重み、MVDR 重み、128 tap 化後の最終重みについて、

\[
w_0[k]^H a[k] \approx 1
\]

\[
w_{\mathrm{MVDR}}[k]^H a[k] \approx 1
\]

\[
\left(w_0[k]-q_{128}[k]\right)^H a[k] \approx 1
\]

を確認する。

### 12.3 補正枝の blocking 性

差分補正重みについて、

\[
q[k]^H a[k] \approx 0
\]

が成り立つことを確認する。

### 12.4 128 tap 近似誤差

IFFT 後に 128 tap 化した補正重みを再 FFT し、

\[
E_q[k] = q[k] - q_{128}[k]
\]

を確認する。

### 12.5 BTR 評価

以下を BTR で確認する。

- 目標方位のレベル低下がないこと。
- 干渉方位の抑圧が得られること。
- 係数更新時刻に縦縞が出ないこと。
- 方位変化時に過剰な追従遅れまたは発振的挙動がないこと。

---

## 13. 基本方式の結論

固定遅延+差分補正MVDR は、以下の式に集約される。

\[
q[k] = w_0[k] - w_{\mathrm{MVDR}}[k]
\]

\[
y[k] = w_0[k]^H X[k] - q[k]^H X[k]
\]

\[
y[k] = w_{\mathrm{MVDR}}[k]^H X[k]
\]

すなわち、固定整相を主経路に残し、MVDR との差分のみを時間領域 128 tap FIR として補正する方式である。

本方式は、固定整相による大きな幾何遅延の吸収と、MVDR による統計的な干渉抑圧を分離できる。これにより、MVDR 重み全体を FIR 化するよりも、短い FIR で実装できる可能性が高い。
