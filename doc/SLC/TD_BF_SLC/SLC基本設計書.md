# SLC基本設計書

## 1. 目的

本書は、固定整相後のビーム出力に対して適用する SLC、すなわちサイドローブキャンセラの基本設計を定義する。

本設計の目的は、固定ビームフォーマのメインローブ形状を維持しつつ、target beam にサイドローブ経由で混入した干渉成分を、guard 外の参照ビーム群から推定して差し引くことである。

本設計では、固定整相処理は変更しない。適応するのは、サイドローブ干渉を推定するためのキャンセラ係数のみである。

---

## 2. やりたいこと

固定整相により、各待ち受け方位に対するビーム出力を得る。

```text
beam_output: [n_beam, n_sample]
```

このうち、通したい方位を target beam とする。target beam に対して、周囲の guard 領域を保護し、それ以外のビームを参照ビームとして使用する。

SLC は、参照ビーム群から target beam に混入した干渉成分を推定し、target beam から差し引く。

```text
固定整相出力
  ↓
target beam d[n]
reference beams u[n]
  ↓
干渉推定 c[n]
  ↓
y[n] = d[n] - c[n]
```

---

## 3. 基本概念図

単一 target の場合の概念図を示す。

```text
固定ビーム出力 beam_output[beam, n]

      target beam
          │
          ▼
        d[n] ───────────────┐
                             │
                             ▼
                         y[n] = d[n] - c[n]
                             ▲
                             │
      guard外 beams          │
          │                  │
          ▼                  │
        u[n] ── adaptive w ──┘
                  │
                  ▼
                c[n]
```

ここで、

```text
d[n] : target beam の固定整相出力
u[n] : 参照ビーム群
w    : 適応キャンセラ係数
c[n] : 干渉推定信号
y[n] : SLC後出力
```

である。

---

## 4. GSC的な考え方

本方式は、古典的な GSC、Generalized Sidelobe Canceller の考え方に近い。

GSC の基本構造は次である。

```text
入力信号
  ├─ 主経路
  │    目標方位を通す固定ビームフォーマ
  │    ↓
  │  d[n]
  │
  └─ 参照経路
       目標方位成分をなるべく含まない参照信号
       ↓
     u[n]
       ↓
     適応キャンセラ
       ↓
     c[n]

出力:
  y[n] = d[n] - c[n]
```

本設計では、チャネル信号から blocking matrix を作るのではなく、固定整相後の guard 外ビームを参照信号として使用する。

したがって、厳密な blocking matrix 型 GSC ではないが、次の構造を持つ。

```text
主経路:
  target beam 出力

参照経路:
  guard 外の固定ビーム出力

適応部:
  参照ビーム群から target beam 内の干渉を推定して差し引く
```

この意味で、本方式はビーム領域 GSC 風 SLC と位置づける。

---

## 5. 基本数式

固定整相後の target beam 出力を、

\[
d[n]
\]

参照ビーム群を、

\[
\mathbf{u}[n]
=
\begin{bmatrix}
u_1[n] \\
u_2[n] \\
\vdots \\
u_K[n]
\end{bmatrix}
\]

とする。

SLC は、参照ビーム群の線形結合により干渉推定信号を作る。

\[
c[n]
=
\mathbf{w}^H\mathbf{u}[n]
\]

SLC 後の出力は、

\[
y[n]
=
d[n]-c[n]
\]

すなわち、

\[
y[n]
=
d[n]-\mathbf{w}^H\mathbf{u}[n]
\]

である。

ここで、\(\mathbf{w}\) は SLC 係数であり、入力信号に応じて更新される。

---

## 6. 共分散と SLC 係数の考え方

SLC 係数は、target beam 出力 \(d[n]\) のうち、参照ビーム群 \(\mathbf{u}[n]\) で説明できる成分を推定する係数である。

参照ビーム群の相関行列を、

\[
\mathbf{R}_{uu}
=
E\{\mathbf{u}[n]\mathbf{u}^H[n]\}
\]

参照ビーム群と target beam の相互相関を、

\[
\mathbf{r}_{ud}
=
E\{\mathbf{u}[n]d^*[n]\}
\]

とする。

平均二乗誤差、

\[
J
=
E\{|d[n]-\mathbf{w}^H\mathbf{u}[n]|^2\}
\]

を最小化する係数は、

\[
\mathbf{R}_{uu}\mathbf{w}
=
\mathbf{r}_{ud}
\]

を満たす。

したがって、

\[
\mathbf{w}
=
\mathbf{R}_{uu}^{-1}\mathbf{r}_{ud}
\]

である。

実用上は、安定化のために対角ロードを入れ、

\[
\mathbf{w}
=
(\mathbf{R}_{uu}+\lambda\mathbf{I})^{-1}\mathbf{r}_{ud}
\]

とする。

---

## 7. guard 領域の役割

SLC で最も重要なのは、参照ビーム群に target 成分を入れないことである。

そのため、target beam の周囲に guard 領域を設ける。

```text
beam index:
... 10  11  12  13  14  15  16 ...

例:
  target = 13
  guard = 2

保護領域:
  11, 12, 13, 14, 15

参照領域:
  ..., 10, 16, ...
```

SLC は guard 外の参照ビームだけを使う。

これにより、target beam 近傍のメインローブ成分を保護し、target 信号が SLC によって自己消去されることを防ぐ。

---

## 8. 複数 target beam

複数方位の信号を通したい場合、target beam を複数持つことができる。

```text
target_beams = [b1, b2, b3, ...]
```

この場合、各 target beam の周囲に guard 領域を作り、その和集合を保護領域とする。

```text
protected region
  = union of all target guard regions
```

参照ビームは、すべての target guard 領域の外側から選ぶ。

```text
reference beams
  = all beams - protected region
```

SLC 出力は target ごとに作る。

\[
y_i[n]
=
d_i[n]-\mathbf{w}_i^H\mathbf{u}[n]
\]

ここで、\(i\) は target 番号である。

参照ビーム群は target 間で共通としてよいが、SLC 係数 \(\mathbf{w}_i\) は target ごとに持つ。

---

## 9. self-nulling と安全側設計

SLC の主な危険は self-nulling、すなわち target 信号の自己消去である。

self-nulling は、参照ビーム群に target 信号が混入し、SLC が target 信号まで干渉として学習してしまうことで発生する。

したがって、本設計では以下を基本方針とする。

```text
1. guard 領域は target 成分を保護するために使う。
2. target 数が増えても、guard を安易に狭めない。
3. 参照ビームが不足する場合は、guard を削らず SLC を弱化または無効化する。
4. SLC が危険な条件では、固定整相出力をそのまま通す。
```

---

## 10. target が多い場合の考え方

target が多くなると、保護領域が広がり、参照ビームが不足する。

このとき、SLC を成立させるために guard を小さくするのは危険である。

guard は固定ビームフォーマのメインローブ保護幅であり、target 数とは独立に決めるべきである。

したがって、target が多い場合は、次の優先順位とする。

```text
1. guard 幅は維持する。
2. 参照ビーム数を確認する。
3. 参照不足なら SLC を弱化する。
4. さらに危険なら SLC を無効化する。
```

---

## 11. SLC 無効化の考え方

SLC は常に有効である必要はない。

条件が悪い場合に無理に SLC を動作させると、干渉抑圧よりも target 信号劣化の方が大きくなる可能性がある。

そのため、以下のような条件では SLC を弱化または無効化する。

```text
参照ビーム数が少なすぎる
共分散推定に必要なサンプル数が不足している
target 数が多すぎて保護領域が広すぎる
SLC 係数が過大になっている
更新が不安定である
```

SLC 無効化時の出力は、固定整相出力そのものとする。

\[
y[n] = d[n]
\]

この設計により、異常時は必ず安全側、すなわち固定整相の性能に戻る。

---

## 12. キャンセル量の弱化

SLC を完全に無効化する前に、キャンセル量を弱める設計も有効である。

\[
y[n]
=
d[n]-\eta\mathbf{w}^H\mathbf{u}[n]
\]

ここで、\(\eta\) はキャンセル強度である。

```text
η = 1.0 : 通常SLC
η = 0.5 : 半分だけキャンセル
η = 0.0 : SLC無効
```

参照不足や不安定な条件では、\(\eta\) を小さくすることで、過剰キャンセルを防ぐ。

---

## 13. 相対方位 SLC と艦首変化

本システムでは、後段処理が相対方位のビーム出力を要求するため、SLC も相対方位系で行う。

ただし、相対方位系では、艦首方位が変わると、同じ外部音源が相対方位上で移動する。

そのため、共分散の忘却平均において、古い艦首方位の統計が現在の相対方位と一致しなくなる場合がある。

そこで、艦首変化量に応じて共分散を速く忘れる制御を導入する。

```text
艦首変化が小さい:
  過去の共分散を長く保持する

艦首変化が大きい:
  古い共分散を速く忘れる
```

これにより、相対方位 SLC でも実用的な共分散学習を行う。

---

## 14. 方位センサ異常時の安全側設計

艦首変化に基づく忘却制御は、方位センサが正常であることを前提とする。

方位センサが異常な場合、艦首変化量が信頼できないため、その値を使って共分散をリセットしたり更新停止したりすると、かえって不安定になる。

したがって、方位センサ異常時は、以下のように安全側に倒す。

```text
方位センサ正常:
  艦首変化を考慮した忘却制御を使う

方位センサ異常:
  艦首変化による制御を無効化する
  通常の時間忘却のみで SLC を継続する
```

SLC 自体は相対方位の固定ビーム出力だけで動作できるため、方位センサ異常時に必ず SLC を無効化する必要はない。

---

## 15. 状態設計

SLC の状態は、概念的に次のように整理できる。

```text
NORMAL:
  SLC有効
  艦首変化連動の忘却制御あり

DEGRADED_HEADING:
  SLC有効
  通常時間忘却のみ
  艦首変化によるリセットなし

LIMITED_REFERENCE:
  SLC弱化
  または更新停止
  またはキャンセル強度 η を低下

DISABLED:
  SLC無効
  固定整相出力をそのまま通す
```

この状態設計により、SLC が効果を出せる条件では有効化し、危険な条件では固定整相出力に戻す。

---

## 16. 片舷処理と両舷処理

実用上は、固定整相を片舷ずつ実施し、その後で両舷のビーム出力を結合する。

SLC は、参照ビーム候補が多いほど性能が出やすいため、理想的には両舷結合後に実施する方が有利である。

ただし、後段処理やシステム制約により、片舷単位で SLC を行う場合もある。

片舷 SLC では参照ビーム数が不足しやすいため、参照ビーム数、target 数、guard 数に基づいて SLC 有効条件を明確に定義する必要がある。

---

## 17. 基本方針まとめ

本 SLC の基本方針は次である。

```text
1. 固定整相後の [n_beam, n_sample] を入力とする。
2. target beam と guard 外 reference beam を分ける。
3. SLC は reference beam から干渉推定信号を作り、target beam から差し引く。
4. 固定整相の遅延や小数遅延 FIR は変更しない。
5. 複数 target beam を持てる。
6. 複数 target の guard 領域は保護領域としてまとめて扱う。
7. guard は target 数に応じて安易に狭めない。
8. 参照不足時は SLC を弱化または無効化する。
9. 相対方位 SLC では、艦首変化に応じて共分散の忘却を制御する。
10. 方位センサ異常時は、艦首変化連動制御を無効化し、通常忘却に戻す。
11. 異常時は固定整相出力を通すことで安全側に倒す。
```
